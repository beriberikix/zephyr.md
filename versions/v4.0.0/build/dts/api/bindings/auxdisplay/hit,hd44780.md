# hit,hd44780

Vendor: [Hitachi Ltd.](../../bindings.md#dt-vendor-hit)

Note

An implementation of a driver matching this compatible is available in
[drivers/auxdisplay/auxdisplay\_hd44780.c](https://github.com/zephyrproject-rtos/zephyr/blob/main/drivers/auxdisplay/auxdisplay_hd44780.c).

## Description

```text
Hitachi HD44780 LCD
```
