# microchip,enc28j60 (on spi bus)

Vendor: [Microchip Technology Inc.](../../bindings.md#dt-vendor-microchip)

Note

An implementation of a driver matching this compatible is available in
[drivers/ethernet/eth\_enc28j60.c](https://github.com/zephyrproject-rtos/zephyr/blob/main/drivers/ethernet/eth_enc28j60.c).

## Description

```text
ENC28J60 standalone 10BASE-T Ethernet controller with SPI interface
```
