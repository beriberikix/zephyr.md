# Central Multilink

[
Browse source code on GitHub
](https://github.com/zephyrproject-rtos/zephyr/blob/main//samples/bluetooth/central_multilink/README.rst/..)

## Overview

Application demonstrating Bluetooth LE Central role functionality by scanning for other
BLE devices and establishing connection to up to 62 peripherals with a strong
enough signal.

## Requirements

- BlueZ running on the host, or
- A board with Bluetooth LE support

## Building and Running

This sample can be found under [samples/bluetooth/central\_multilink](https://github.com/zephyrproject-rtos/zephyr/blob/main/samples/bluetooth/central_multilink)
in the Zephyr tree.

See [Bluetooth](../bluetooth.md#bluetooth) samples for details.
