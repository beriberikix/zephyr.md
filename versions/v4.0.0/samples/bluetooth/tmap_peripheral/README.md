# Telephone and Media Audio Profile (TMAP) Peripheral

[
Browse source code on GitHub
](https://github.com/zephyrproject-rtos/zephyr/blob/main//samples/bluetooth/tmap_peripheral/README.rst/..)

## Overview

Application demonstrating the TMAP peripheral functionality. Implements the CT and UMR roles.

## Requirements

- A board with Bluetooth Low Energy 5.2 support

## Building and Running

This sample can be found under
[samples/bluetooth/tmap\_peripheral](https://github.com/zephyrproject-rtos/zephyr/blob/main/samples/bluetooth/tmap_peripheral) in the Zephyr tree.

See [Bluetooth](../bluetooth.md#bluetooth) samples for details.
