# Central / GATT Write

[
Browse source code on GitHub
](https://github.com/zephyrproject-rtos/zephyr/blob/main//samples/bluetooth/central_gatt_write/README.rst/..)

## Overview

Similar to the [Central](../central/README.md#ble_central "Implement basic Bluetooth LE Central role functionality (scanning and connecting).") sample, except that this
application use GATT Write Without Response.

## Requirements

- BlueZ running on the host, or
- A board with Bluetooth LE support

## Building and Running

This sample can be found under [samples/bluetooth/central\_gatt\_write](https://github.com/zephyrproject-rtos/zephyr/blob/main/samples/bluetooth/central_gatt_write)
in the Zephyr tree.

See [Bluetooth](../bluetooth.md#bluetooth) samples for details.
