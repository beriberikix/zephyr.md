# Degu Evaluation Kit

Board Overview

[![../../../../_images/degu_evk.jpg](../../../../_images/degu_evk.jpg)
](../../../../_images/degu_evk.jpg)

Degu Evaluation Kit

Vendor:
:   Atmark Techno, Inc.

Architecture:
:   arm

SoC:
:   nrf52840

The Degu Evaluation Kit is an IoT device for sensors and actuators
and can connect to an OpenThread mesh network. The Kit has a
Nordic nRF52840 SoC and NXP A71CH Secure Element, and can
connect some Seeed Grove sensors via GPIO, ADC, I2C,
and UART.

For more information, see the [Degu Project website](https://open-degu.com/en/).
