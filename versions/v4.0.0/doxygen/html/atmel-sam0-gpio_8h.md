## [◆ ](#a3aef3839686a04a080b1f2243c5feba3)SAM0\_GPIO\_DEBOUNCE

| #define SAM0\_GPIO\_DEBOUNCE   (1U << 8) |
| --- |

Enable GPIO pin debounce.

The debounce flag is a Zephyr specific extension of the standard GPIO flags specified by the Linux GPIO binding. Only applicable for Atmel SAM0 SoCs.
