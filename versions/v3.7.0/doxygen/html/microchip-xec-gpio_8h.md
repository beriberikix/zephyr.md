**Value:**

gpio\_bank gpio\_bitpos

Microchip XEC GPIO bank and bit position convenience defines.

Microchip XEC documentation uses octal GPIO pin numbering. These macros do not require the user to do octal arithmetic to derive the pin's bit position.

Example DT usage:

gpios = <MCHP\_GPIO\_DECODE\_176 GPIO\_ACTIVE\_HIGH>;
