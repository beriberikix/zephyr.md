# Bluetooth Mesh Profile

The Bluetooth Mesh profile adds secure wireless multi-hop communication for
Bluetooth Low Energy. This module implements the
[Bluetooth Mesh Protocol Specification v1.1](https://www.bluetooth.com/specifications/specs/mesh-protocol/).

Read more about Bluetooth Mesh on the
[Bluetooth SIG Website](https://www.bluetooth.com/bluetooth-resources/?tags=mesh).
