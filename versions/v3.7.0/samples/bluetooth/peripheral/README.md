# Bluetooth: Peripheral

## Overview

Application demonstrating the BLE Peripheral role. It has several well-known and
vendor-specific GATT services that it exposes.

## Requirements

- BlueZ running on the host, or
- A board with BLE support

## Building and Running

This sample can be found under [samples/bluetooth/peripheral](https://github.com/zephyrproject-rtos/zephyr/blob/main/samples/bluetooth/peripheral) in the
Zephyr tree.

See [bluetooth samples section](../bluetooth.md#bluetooth-samples) for details.
