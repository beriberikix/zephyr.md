# Bluetooth: Central / GATT Write

## Overview

Similar to the [Central](../central/README.md#bluetooth-central) sample, except that this
application use GATT Write Without Response.

## Requirements

- BlueZ running on the host, or
- A board with BLE support

## Building and Running

This sample can be found under [samples/bluetooth/central\_gatt\_write](https://github.com/zephyrproject-rtos/zephyr/blob/main/samples/bluetooth/central_gatt_write)
in the Zephyr tree.

See [bluetooth samples section](../bluetooth.md#bluetooth-samples) for details.
