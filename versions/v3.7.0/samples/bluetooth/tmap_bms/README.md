# Bluetooth: TMAP BMS

## Overview

Application demonstrating the LE Audio TMAP Broadcast Media Sender functionality.
Implements the BMS role.

## Requirements

- A board with Bluetooth Low Energy 5.2 support

## Building and Running

This sample can be found under [samples/bluetooth/tmap\_bms](https://github.com/zephyrproject-rtos/zephyr/blob/main/samples/bluetooth/tmap_bms) in the Zephyr tree.

See [bluetooth samples section](../bluetooth.md#bluetooth-samples) for details.
