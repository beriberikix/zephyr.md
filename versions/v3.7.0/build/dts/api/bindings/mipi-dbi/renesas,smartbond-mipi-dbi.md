# renesas,smartbond-mipi-dbi

Vendor: [Renesas Electronics Corporation](../../bindings.md#dt-vendor-renesas)

## Description

These nodes are “mipi-dbi” bus nodes.

```text
Renesas Smartbond(tm) MIPI DBI Host
```
