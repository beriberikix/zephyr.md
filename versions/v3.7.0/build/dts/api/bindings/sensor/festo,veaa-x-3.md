# festo,veaa-x-3

Vendor: [Festo SE & Co. KG](../../bindings.md#dt-vendor-festo)

## Description

```text
Festo VEAA proportional pressure regulator using 3/3-way valve.
Only voltage controlled valve is supported.
```
