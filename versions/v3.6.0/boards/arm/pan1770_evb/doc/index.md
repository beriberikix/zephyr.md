# PAN1770 Evaluation Board

## Overview

The PAN1770 Evaluation Board is a development tool for the PAN1770 module
which is based on the nRF52840 chipset from Nordic Semiconductor.

It is basically a clone of the official nRF52840 development kit (PCA10056)
from Nordic Semiconductor. Please refer to [nRF52840 DK](../../nrf52840dk_nrf52840/doc/index.md#nrf52840dk-nrf52840) for
further information.

![PAN1770 Evaluation Board](../../../../_images/pan1770_evaluation_board.jpg)

You can find more information about the PAN1770 module and the PAN1770
evaluation board on the [product website](https://industry.panasonic.eu/products/devices/wireless-connectivity/bluetooth-low-energy-modules/pan1770-nrf52840) [[1]](#id1).

The PAN1770 evaluation board is closely linked to these other evaluation
boards:

- [PAN1780 Evaluation Board](../../pan1780_evb/doc/index.md#pan1780-evb)
- [PAN1781 Evaluation Board](../../pan1781_evb/doc/index.md#pan1781-evb)
- [PAN1782 Evaluation Board](../../pan1782_evb/doc/index.md#pan1782-evb)

## Usage

You can find the [user guide](https://pideu.panasonic.de/development-hub/pan1770/evaluation_board/user_guide/) [[3]](#id6) for the PAN1770 Evaluation Board in the
[Panasonic Wireless Connectivity Development Hub](https://pideu.panasonic.de/development-hub/) [[2]](#id3).

The user guide contains (amongst other things) detailed information about

- pin mapping
- powering options
- breakout pin header interface
- current consumption measurement
- software development

and other things.

The schematics for the PAN1770 Evaluation Board are available in the
[download section](https://pideu.panasonic.de/development-hub/pan1770/downloads/) [[4]](#id8) of the [Panasonic Wireless Connectivity Development Hub](https://pideu.panasonic.de/development-hub/) [[2]](#id3).
