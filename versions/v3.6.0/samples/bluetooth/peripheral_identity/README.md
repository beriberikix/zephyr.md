# Bluetooth: Peripheral Identity

## Overview

This sample demonstrates use of multiple identity and the ability to be
connected to from multiple central devices.

## Requirements

- BlueZ running on the host, or
- A board with BLE support

## Building and Running

This sample can be found under [samples/bluetooth/peripheral\_identity](https://github.com/zephyrproject-rtos/zephyr/blob/main/samples/bluetooth/peripheral_identity)
in the Zephyr tree.

See [bluetooth samples section](../bluetooth.md#bluetooth-samples) for details.
