# Bluetooth: TMAP Central

## Overview

Application demonstrating the LE Audio TMAP central functionality. Implements the CG and UMS roles.

## Requirements

- A board with Bluetooth Low Energy 5.2 support

## Building and Running

This sample can be found under
[samples/bluetooth/tmap\_central](https://github.com/zephyrproject-rtos/zephyr/blob/main/samples/bluetooth/tmap_central) in the Zephyr tree.

See [bluetooth samples section](../bluetooth.md#bluetooth-samples) for details.
