# Bluetooth Qualification

## Qualification Listings

The Zephyr BLE stack has obtained qualification listings for both the Host
and the Controller.
See the tables below for a list of qualification listings

### Host qualifications

| Zephyr version | Link | Qualifying Company |
| --- | --- | --- |
| 2.2.x | [QDID 151074](https://launchstudio.bluetooth.com/ListingDetails/109287) | Demant A/S |
| 1.14.x | [QDID 139258](https://launchstudio.bluetooth.com/ListingDetails/95152) | The Linux Foundation |
| 1.13 | [QDID 119517](https://launchstudio.bluetooth.com/ListingDetails/70189) | Nordic Semiconductor |

### Mesh qualifications

| Zephyr version | Link | Qualifying Company |
