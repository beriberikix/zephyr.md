# Bluetooth: TMAP BMR

## Overview

Application demonstrating the LE Audio TMAP Broadcast Media Receiver functionality.
Implements the BMR role.

## Requirements

- A board with Bluetooth Low Energy 5.2 support

## Building and Running

This sample can be found under [samples/bluetooth/tmap\_bmr](https://github.com/zephyrproject-rtos/zephyr/blob/main/samples/bluetooth/tmap_bmr) in the Zephyr tree.

See [bluetooth samples section](../bluetooth.md#bluetooth-samples) for details.
