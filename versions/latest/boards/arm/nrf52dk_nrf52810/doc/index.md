# nRF52810 emulation on nRF52 DK

## Overview

The nrf52dk\_nrf52810 board is a modified version of the [nRF52 DK](../../nrf52dk_nrf52832/doc/index.md#nrf52dk-nrf52832)
that enforces the limitations imposed by the nRF52810 IC, which is a
cost-reduced variant of the original nRF52832. Since Nordic does not offer a
development kit for the nRF52810 you can use this board to develop for this
IC while using the nRF52 Development Kit (PCA10040).

See [nRF52 DK](../../nrf52dk_nrf52832/doc/index.md#nrf52dk-nrf52832) for more information about the development board and
[nRF52810 website](https://www.nordicsemi.com/Products/Low-power-short-range-wireless/nRF52810) [[1]](#id1) for the official reference on the IC itself.
