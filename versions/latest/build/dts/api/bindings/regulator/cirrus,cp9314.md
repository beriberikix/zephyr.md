# cirrus,cp9314 (on i2c bus)

Vendor: [Cirrus Logic, Inc.](../../bindings.md#dt-vendor-cirrus)

## Description

```text
Cirrus CP9314 Buck Switched Cap DC/DC Converter

converter@72 {
  compatible = "cirrus,cp9314";
  reg = <0x72>;

  cirrus,initial-switched-capacitor-mode = "2:1";
};
```
