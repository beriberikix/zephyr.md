## Detailed Description

Kernel Spin Lock.

This struct defines a spin lock record on which CPUs can wait with [k\_spin\_lock()](group__spinlock__apis.md#gaac60da4347f5b29ff8c4e5f24c99b3bf "Lock a spinlock."). Any number of spinlocks may be defined in application code.
