# nxp,mipi-dsi-2l

Vendor: [NXP Semiconductors](../../bindings.md#dt-vendor-nxp)

Note

An implementation of a driver matching this compatible is available in
[drivers/mipi\_dsi/dsi\_mcux\_2l.c](https://github.com/zephyrproject-rtos/zephyr/blob/main/drivers/mipi_dsi/dsi_mcux_2l.c).

## Description

These nodes are “mipi-dsi” bus nodes.

```text
NXP MCUX MIPI DSI 2L
```
