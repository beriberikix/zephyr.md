# infineon,xmc4xxx-ethernet

Vendor: [Infineon Technologies](../../bindings.md#dt-vendor-infineon)

[drivers/ethernet/eth\_xmc4xxx.c](https://github.com/zephyrproject-rtos/zephyr/blob/main/drivers/ethernet/eth_xmc4xxx.c).

