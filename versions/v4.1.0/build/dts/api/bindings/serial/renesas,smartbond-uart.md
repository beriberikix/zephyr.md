# renesas,smartbond-uart

Vendor: [Renesas Electronics Corporation](../../bindings.md#dt-vendor-renesas)

Note

An implementation of a driver matching this compatible is available in
[drivers/serial/uart\_smartbond.c](https://github.com/zephyrproject-rtos/zephyr/blob/main/drivers/serial/uart_smartbond.c).

## Description

These nodes are “uart” bus nodes.

```text
Renesas SmartBond(tm) UART
```
