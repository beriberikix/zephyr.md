# aptina,mt9m114 (on i2c bus)

Vendor: [Aptina Imaging](../../bindings.md#dt-vendor-aptina)

Note

[drivers/video/mt9m114.c](https://github.com/zephyrproject-rtos/zephyr/blob/main/drivers/video/mt9m114.c).

