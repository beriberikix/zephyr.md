# ti,bq25180 (on i2c bus)

Vendor: [Texas Instruments](../../bindings.md#dt-vendor-ti)

Note

An implementation of a driver matching this compatible is available in
[drivers/charger/charger\_bq25180.c](https://github.com/zephyrproject-rtos/zephyr/blob/main/drivers/charger/charger_bq25180.c).

## Description

```text
BQ25180 I2C Controlled, 1-Cell, 1-A Linear Battery Charger with Power Path
and Ship Mode.

The device has a single child node for the charger. For example:

bq25180@6a {
  compatible = "ti,bq25180";
  reg = <0x6a>;

  constant-charge-current-max-microamp = <500000>;
};
```
