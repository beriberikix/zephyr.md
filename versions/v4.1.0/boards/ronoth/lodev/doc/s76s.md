# AcSIP S76S

## Overview

The [AcSIP S76S](http://www.acsip.com.tw/index.php?action=products-detail&fid1=11&fid2=29&fid3=27&id=79&lang=3) is an SiP from [AcSIP](http://www.acsip.com.tw) containing an [STMicro STM32L073RZ](https://www.st.com/en/microcontrollers-microprocessors/stm32l073rz.html) MCU,
a [Semtech SX1276](https://www.semtech.com/products/wireless-rf/lora-transceivers/sx1276) LoRaWAN transceiver, and a +20 dBm power amplifier. Refer to the
[AcSIP S76S Product Information Brief](https://www.acsip.com.tw/index.php?action=download_pro&perm=d&id=365) for details. Further information is available
from the [AcSIP Product Data Download](http://www.acsip.com.tw/index.php?action=technical) site (may need login). The parts are [available through TechShip](https://techship.com/products/acsip-lorawan-module-s76s/).

Zephyr applications may use the **acsip\_s76s** configuration to use this SiP.

![AcSIP S76S system on a chip, containing STMicro STM32L073RZ and Semtech SX1276](../../../../_images/acsip_s76s.jpg)

AcSIP S76S

## Device Resources

The embedded [STMicro STM32L073RZ](https://www.st.com/en/microcontrollers-microprocessors/stm32l073rz.html) has some GPIOs and SPI2 internally committed to the LoRaWAN
transceiver operation. See [internally committed table](#internally-committed-table) for a list of resources committed to this function.

As a result, some functions, ports and features of the [STMicro STM32L073RZ](https://www.st.com/en/microcontrollers-microprocessors/stm32l073rz.html) are not available.
See [unavailable table](#unavailable-table) for a list of resources not available due to a lack of pin-out.

Available pinned-out resources are listed in [this table](#this-table). The actual [S76S pinout table](#s76s-pinout-table) is below.

These tables are STM32L07x generic - some pinned-out resources may be unavailable due to limitations
on the [STMicro STM32L073RZ](https://www.st.com/en/microcontrollers-microprocessors/stm32l073rz.html) processor itself. Consult with the [STMicro STM32L073RZ](https://www.st.com/en/microcontrollers-microprocessors/stm32l073rz.html) documentation.

## Development

The [Ronoth](https://ronoth.com/) [LoDev](https://ronoth.com/products/lodev-s76s-lora-soc-development-board?variant=31608819417220) board is an open source development board, see its board description files.

## Debugging

Programming and debugging uses the SWD port, as on any STM32 processor. An ST-LINK/V2 adapter
may be used with the appropriate software (*st-utils* package on Linux).
