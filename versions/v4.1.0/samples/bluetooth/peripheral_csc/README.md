# Cycling Speed and Cadence (CSC) Peripheral

[
Browse source code on GitHub
](https://github.com/zephyrproject-rtos/zephyr/blob/main//samples/bluetooth/peripheral_csc/README.rst/..)

## Overview

Similar to the [Peripheral](../peripheral/README.md#ble_peripheral "Implement basic Bluetooth LE Peripheral role functionality (advertising and exposing GATT services).") sample, except that this
application specifically exposes the CSC (Cycling Speed and Cadence) GATT
Service.

## Requirements

- BlueZ running on the host, or
- A board with Bluetooth LE support

## Building and Running

This sample can be found under [samples/bluetooth/peripheral\_csc](https://github.com/zephyrproject-rtos/zephyr/blob/main/samples/bluetooth/peripheral_csc) in the
Zephyr tree.

See [Bluetooth](../bluetooth.md#bluetooth) samples for details.
