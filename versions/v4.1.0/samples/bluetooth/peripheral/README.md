# Peripheral

[
Browse source code on GitHub
](https://github.com/zephyrproject-rtos/zephyr/blob/main//samples/bluetooth/peripheral/README.rst/..)

## Overview

Application demonstrating the Bluetooth LE Peripheral role. It has several well-known and
vendor-specific GATT services that it exposes.

## Requirements

- BlueZ running on the host, or
- A board with Bluetooth LE support

## Building and Running

This sample can be found under [samples/bluetooth/peripheral](https://github.com/zephyrproject-rtos/zephyr/blob/main/samples/bluetooth/peripheral) in the
Zephyr tree.

See [Bluetooth](../bluetooth.md#bluetooth) samples for details.
