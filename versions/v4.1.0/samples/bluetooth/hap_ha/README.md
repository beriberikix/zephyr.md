# Hearing Access Profile (HAP) Hearing Aid (HA)

[
Browse source code on GitHub
](https://github.com/zephyrproject-rtos/zephyr/blob/main//samples/bluetooth/hap_ha/README.rst/..)

## Overview

Application demonstrating the LE Audio Hearing Aid sample functionality.
Starts advertising and awaits connection from a Hearing Aid Unicast Client (HAUC)
or Hearing Aid Remote Controller (HARC).

## Requirements

- BlueZ running on the host, or
- A board with Bluetooth Low Energy 5.2 support

## Building and Running

This sample can be found under
[samples/bluetooth/hap\_ha](https://github.com/zephyrproject-rtos/zephyr/blob/main/samples/bluetooth/hap_ha) in the Zephyr tree.

See [Bluetooth](../bluetooth.md#bluetooth) samples for details.
